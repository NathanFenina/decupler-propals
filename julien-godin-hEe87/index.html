🧠 Le cerveau Claude Code

Ce que tu pilotes après livraison.

4 briques internes : skills réutilisables, mémoires persistantes, APIs externes, terminal de pilotage. Tu n'as pas besoin d'être dev — tu parles en français.

Brique 1

Skills de rédaction

Compétences nommées (ex: create-city-page, verify-seo, publish-batch) qui encapsulent la trame SEO. Tu invoques par mots-clés et l'agent enchaîne validation volume → rédaction → image → publication.

Brique 2

Mémoires .md structurées

Deux fichiers persistants : memory_business.md (ton positionnement, ta promesse, ton ton, tes coachs partenaires) et memory_pages.md (la trame d'une page-type, les variables, les règles d'unicité). Cohérence garantie sur 280 pages.

Brique 3

Pipeline d'APIs externes

Claude pour le contenu textuel. OpenAI ou Gemini pour 2 images distinctes par page. Data For SEO pour valider le volume avant de produire — pas de page mort-née.

Brique 4

Interface terminal

Tu ouvres une conv Claude Code et tu décides : "publie 5 pages cette semaine", "audite les 30 dernières", "ajoute la Belgique", "génère un rapport SEO". Tu pilotes le rythme — daily, weekly, monthly.
